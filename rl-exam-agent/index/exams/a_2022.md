## Q1 (30 pts) — Job-search MDP and the threshold "work" policy
**Topics:** MDP formalism, optimal policy structure, threshold policy, Q-function algebra | **Pillar:** Planning | **Difficulty:** 4
**Maps to:** lecture_02, lecture_03
**Statement (English translation):**
A sequential decision problem. There is a single worker and $n$ jobs; the wage of job $i$ is $r_i > 0$. At the beginning of each month:
- If the worker is **employed** at job $i$, he chooses one of:
  1. **work** — earns wage $r_i$; at month's end he is fired with probability $\alpha$ and becomes unemployed, and with probability $1-\alpha$ stays employed at job $i$.
  2. **no-work** — earns no wage this month and becomes unemployed.
- If the worker is **unemployed**, he receives an offer to work at one of the $n$ jobs; the offer is job $i$ with probability $1/n$. Given the offer for job $i$ he chooses one of:
  1. **work** — same dynamics as employed-at-$i$: earns $r_i$, then fired w.p. $\alpha$ (becomes unemployed), stays employed at $i$ w.p. $1-\alpha$.
  2. **no-work** — stays unemployed and earns no wage this month.

At the start ($t=0$) the worker is employed at job 1. Let $\pi^*$ be the optimal policy for the discounted return with discount $0<\gamma<1$, with value functions $Q^*, V^*$; $r(s,a)$ is the immediate reward.

(a) Write the MDP formally ($S, A, s_0, R, P$). **Hint:** it can be written with two actions and $n$ states.

(b) Show there exists a value $\Phi^*$ such that $Q^*(s, \text{no-work}) = \gamma\Phi^*$ for **every** state $s$.

(c) Given two states $s_1, s_2$ with $\pi^*(s_1)=\text{work}$ and $\pi^*(s_2)=\text{no-work}$, show $Q^*(s_1,\text{work}) \ge Q^*(s_2,\text{work})$.

(d) Show there exists a threshold $\Psi$ such that for every state $s$: if $\Psi < r(s,\text{work})$ then $\pi^*(s)=\text{work}$, and if $\Psi > r(s,\text{work})$ then $\pi^*(s)=\text{no-work}$. **Hint:** first show $Q^*(s,\text{work}) \ge \frac{1}{1-\gamma(1-\alpha)}\big[r(s,\text{work}) + \gamma\alpha\Phi^*\big]$.

**Solution sketch:**

**(a)** Collapse to $n$ states because both "employed at $i$" and "unemployed with offer $i$" share the same action set and dynamics. $S=\{1,\dots,n\}$ (current job $i$), $A=\{\text{work},\text{no-work}\}$, $s_0=1$. Rewards: $r(i,\text{work})=r_i$, $r(i,\text{no-work})=0$. Transitions: under **work**, $P(i\!\to\! i)=(1-\alpha)+\alpha/n$ and $P(i\!\to\! j)=\alpha/n$ for $j\neq i$ (stay w.p. $1-\alpha$; else fired and re-offered a uniform job). Under **no-work**, $P(i\!\to\! j)=1/n$ for all $j$ (become unemployed, uniform re-offer).

**(b)**

$$Q^*(s,\text{no-work}) = 0 + \gamma\sum_{j}\tfrac1n V^*(j) = \gamma\,\Phi^*$$

with $\Phi^* := \tfrac1n\sum_j V^*(j) = \mathbb{E}_{j\sim\text{unif}}[V^*(j)]$ — independent of $s$ since no-work sends every state to the same uniform next-state distribution.

**(c)**

$$\pi^*(s_1)=\text{work}\Rightarrow Q^*(s_1,\text{work})\ge Q^*(s_1,\text{no-work})=\gamma\Phi^*$$

$$\pi^*(s_2)=\text{no-work}\Rightarrow \gamma\Phi^*=Q^*(s_2,\text{no-work})\ge Q^*(s_2,\text{work})$$

Chain:

$$Q^*(s_1,\text{work})\ge\gamma\Phi^*\ge Q^*(s_2,\text{work})$$

**(d)**

$$Q^*(s,\text{work}) = r_s + \gamma[(1-\alpha)V^*(s)+\alpha\Phi^*]$$

with $r_s:=r(s,\text{work})$. Since $V^*(s)\ge Q^*(s,\text{work})$, substitute to get $Q^*(s,\text{work})[1-\gamma(1-\alpha)] \ge r_s+\gamma\alpha\Phi^*$, i.e. the hint bound (equality when work is optimal, since then $V^*(s)=Q^*(s,\text{work})$). Work is optimal iff $Q^*(s,\text{work})\ge\gamma\Phi^*$. Plugging the equality form $Q^*(s,\text{work})=\frac{r_s+\gamma\alpha\Phi^*}{1-\gamma(1-\alpha)}$ and solving $\ge\gamma\Phi^*$ gives $r_s \ge \gamma(1-\alpha)(1-\gamma)\Phi^*$. Hence the threshold

$$\boxed{\Psi=\gamma(1-\alpha)(1-\gamma)\,\Phi^*}$$

is independent of $s$ (depends only on global $\gamma,\alpha,\Phi^*$), so $\pi^*(s)=\text{work}\iff r(s,\text{work})>\Psi$. Verified numerically (random $n=6$ instance, $\gamma=0.9,\alpha=0.3$): the threshold $\Psi$ separates work/no-work exactly and the $Q^*$ identity holds where work is optimal.

**💡 Trick.** "No-work" resets every state to the same uniform next-state distribution, so $Q^*(s,\text{no-work})=\gamma\Phi^*$ is a single state-independent constant — the optimal decision reduces to comparing $r(s,\text{work})$ against one global threshold.

**⚠️ Watch out.** The clean form $Q^*(s,\text{work})=\frac{r_s+\gamma\alpha\Phi^*}{1-\gamma(1-\alpha)}$ is an *equality* only where work is optimal (there $V^*(s)=Q^*(s,\text{work})$); using it as an equality everywhere, instead of the inequality $V^*(s)\ge Q^*(s,\text{work})$, breaks the argument.

## Q2 (30 pts) — POMDP: belief update and finite-horizon value iteration
**Topics:** POMDP, belief state, Bayesian filtering, value iteration, horizon | **Pillar:** Planning | **Difficulty:** 4
**Maps to:** lecture_11, lecture_03
**Statement (English translation):**
A POMDP with two states, three actions; the **observation is the immediate reward**.
$S=\{s_1,s_2\}$, $A=\{a_1,a_2,a_3\}$. Transitions: $p(s\mid s,a_1)=p(s\mid s,a_2)=1$ for all $s$ (actions $a_1,a_2$ are the identity); $p(s_1\mid s_2,a_3)=p(s_2\mid s_2,a_3)=p(s_2\mid s_1,a_3)=p(s_1\mid s_1,a_3)=1/2$ ($a_3$ jumps to a uniform next state).
Reward $r(s,a)=1$ if $(s,a)\in\{(s_1,a_1),(s_2,a_2)\}$; $=2/3$ if $a=a_3$; $=0$ otherwise.
Observations $O=\{0,2/3,1\}$; nonzero $Ob(\cdot\mid s',a)$ (with $s'$ the **next** state): $Ob(1\mid s_1,a_1)=Ob(1\mid s_2,a_2)=1$; $Ob(0\mid s_1,a_2)=Ob(0\mid s_2,a_1)=1$; $Ob(2/3\mid s_1,a_3)=Ob(2/3\mid s_2,a_3)=1$.
Let $x$ = probability of being in $s_1$ (belief).

(a) Compute the belief-update posterior in the cases $P(s_1\mid a_1,o=1,x)$, $P(s_1\mid a_1,o=0,x)$, $P(s_1\mid a_3,o=2/3,x)$.

(b) Assume horizon $H=1$. (b1) Compute $V_1^*(x\mid a_i)$ for each action. (b2) Compute $V_1^*(x)$.

(c) For $H=2$, compute $V_2^*(x)$. **Hint:** $V_2^*(x\mid a_1)=V_2^*(1-x\mid a_2)$.

**Solution sketch:**
Bayesian belief update $b'(s')\propto Ob(o\mid s',a)\sum_s b(s)p(s'\mid s,a)$.

**(a)** Under $a_1$ (identity) the observation reveals the state: $P(s_1\mid a_1,o=1,x)=1$, $P(s_1\mid a_1,o=0,x)=0$. Under $a_3$ the observation is always $2/3$ and the next state is uniform: $P(s_1\mid a_3,o=2/3,x)=1/2$.

**(b)** Expected immediate reward under belief $x$: $V_1^*(x\mid a_1)=x$, $V_1^*(x\mid a_2)=1-x$, $V_1^*(x\mid a_3)=2/3$. So

$$V_1^*(x)=\max(x,\,1-x,\,2/3)$$

(c) One-step lookahead $V_2^*(x\mid a)=R(x,a)+\sum_o P(o\mid x,a)\,V_1^*(b')$. Under $a_1$: obs reveals state so next belief is $0$ or $1$, and $V_1^*(0)=V_1^*(1)=1$, giving $V_2^*(x\mid a_1)=x+1$. By the hint symmetry $V_2^*(x\mid a_2)=2-x$. Under $a_3$: belief becomes $1/2$, $V_1^*(1/2)=2/3$, so $V_2^*(x\mid a_3)=2/3+2/3=4/3$. Therefore

$$V_2^*(x)=\max(x+1,\,2-x,\,4/3)=1+\max(x,1-x)$$

**Verified numerically:** $a_3$ is **never** strictly optimal at $H=2$ (its value $4/3$ is below $\max(x+1,2-x)\ge 3/2$ everywhere), and the hint identity $V_2^*(x\mid a_1)=V_2^*(1-x\mid a_2)$ holds.

**💡 Trick.** Because the observation *is* the reward, $a_1$ fully reveals the state (posterior collapses to $0$ or $1$), so its lookahead value is exact: next-step value is $1$ regardless, giving $V_2^*(x\mid a_1)=x+1$ and, by symmetry, $V_2^*(x\mid a_2)=2-x$.

**⚠️ Watch out.** Don't confuse the immediate reward $2/3$ of $a_3$ with a good long-run choice — at $H=2$ its total $4/3$ never beats $\max(x+1,2-x)\ge3/2$, so $a_3$ is never strictly optimal.

## Q3 (20 pts) — Quadratic-linear policy: representability, score function, REINFORCE
**Topics:** parametric policy, policy gradient, score function, REINFORCE | **Pillar:** Approximation | **Difficulty:** 3
**Maps to:** lecture_09, lecture_10
**Statement (English translation):**
An MDP where each state $s$ and action $a$ has a feature vector $\phi(s,a)\in\mathbb{R}^d$ ($S,A$ finite). A policy is parameterized by $\theta\in\mathbb{R}^d$, $\theta\neq 0$, with
$$\pi(a\mid s;\theta)=\frac{(\phi(s,a)\cdot\theta)^2}{\sum_{a'}(\phi(s,a')\cdot\theta)^2}.$$

(a) In state $s$ there are three actions with $\phi(s,a_1)=(1,-1),\ \phi(s,a_2)=(-1,1),\ \phi(s,a_3)=(1,1)$.

 (a1) Is there $\theta$ with $\pi(a_2\mid s;\theta)=\pi(a_1\mid s;\theta)=0$ and $\pi(a_3\mid s;\theta)=1$? (yes/no; give $\theta$ or explain).

 (a2) Is there, for every distribution $(p_1,p_2,p_3)$, a $\theta$ with $\pi(a_i\mid s;\theta)=p_i$? (yes/no; give a counterexample or explain).

(b) Compute $\nabla_\theta\log\pi(a\mid s;\theta)$ (explain).

(c) Write the REINFORCE update.

**Solution sketch:**

(a1) **Yes.** Need $\phi(s,a_1)\cdot\theta=\theta_1-\theta_2=0$ and $\phi(s,a_2)\cdot\theta=-\theta_1+\theta_2=0$ (both give $\theta_1=\theta_2$) with $\phi(s,a_3)\cdot\theta=\theta_1+\theta_2\neq0$. E.g. $\theta=(1,1)$: numerators $(0,0,4)$, so $\pi=(0,0,1)$.

(a2) **No.** Since $\phi(s,a_1)=-\phi(s,a_2)$, we always have $(\phi(s,a_1)\cdot\theta)^2=(\phi(s,a_2)\cdot\theta)^2$, hence $\pi(a_1\mid s;\theta)=\pi(a_2\mid s;\theta)$ for every $\theta$. Any target with $p_1\neq p_2$ (e.g. $(1,0,0)$) is unrepresentable.

**(b)** With $Z=\sum_{a'}(\phi(s,a')\cdot\theta)^2$: $\log\pi = 2\log|\phi(s,a)\cdot\theta| - \log Z$, so

$$\begin{aligned}\nabla_\theta\log\pi(a\mid s;\theta)&=\frac{2\,\phi(s,a)}{\phi(s,a)\cdot\theta}-\frac{2\sum_{a'}(\phi(s,a')\cdot\theta)\,\phi(s,a')}{\sum_{a'}(\phi(s,a')\cdot\theta)^2}\\ &=\frac{2\phi(s,a)}{\phi(s,a)\cdot\theta}-\mathbb{E}_{a'\sim\pi}\!\Big[\frac{2\phi(s,a')}{\phi(s,a')\cdot\theta}\Big].\end{aligned}$$

**(c)** REINFORCE: sample a trajectory, then for each $t$ update $\theta\leftarrow\theta+\eta_t\,\gamma^t G_t\,\nabla_\theta\log\pi(a_t\mid s_t;\theta)$, where $G_t=\sum_{k\ge t}\gamma^{k-t}r_k$ is the return-to-go and $\nabla_\theta\log\pi$ is the score above.

**💡 Trick.** With a squared-linear policy $\log\pi=2\log|\phi(s,a)\cdot\theta|-\log Z$, so the score is $\frac{2\phi(s,a)}{\phi(s,a)\cdot\theta}$ minus its $\pi$-average — the familiar "feature minus expected feature" form. For (a2), $\phi(s,a_1)=-\phi(s,a_2)$ forces $\pi(a_1)=\pi(a_2)$ always, killing any target with $p_1\neq p_2$.

**⚠️ Watch out.** The square means the policy sees $|\phi\cdot\theta|$, not its sign, so opposite feature vectors are indistinguishable; also the score blows up where $\phi(s,a)\cdot\theta=0$, so don't treat it as defined at a zero-numerator action.

## Q4 (20 pts) — A max-over-all Bellman-type operator: contraction and fixed point
**Topics:** contraction mapping, sup-norm, Bellman operator, fixed point | **Pillar:** Planning | **Difficulty:** 3
**Maps to:** lecture_03, lecture_02
**Statement (English translation):**
MDP $M=(S,A,p,s_0,r)$. For $U\in\mathbb{R}^{|S\times A|}$ and $0<\gamma<1$ define
$$(TU)(s,a)=r(s,a)+\gamma\max_{a',s'}U(s',a').$$

(a) State the definition of $H$ being $\gamma$-contracting in $\|\cdot\|_\infty$.

(b) Prove $T$ is $\gamma$-contracting in $\|\cdot\|_\infty$.

(c) What is the fixed point $U^*$ of $T$? For every $(s,a)$ give the value $T$ converges to, as a function of the MDP parameters only. **Hint:** start from the $(s,a)$ with maximal reward.

**Solution sketch:**

**(a)** $H$ is $\gamma$-contracting in $\|\cdot\|_\infty$ if $\|HU-HV\|_\infty\le\gamma\|U-V\|_\infty$ for all $U,V$.

**(b)** The $r(s,a)$ term cancels:

$$(TU)(s,a)-(TV)(s,a)=\gamma\big[\max_{s',a'}U(s',a')-\max_{s',a'}V(s',a')\big]$$

Using $|\max f-\max g|\le\max|f-g|$, this is $\le\gamma\max_{s',a'}|U(s',a')-V(s',a')|=\gamma\|U-V\|_\infty$, uniformly in $(s,a)$; take the max over $(s,a)$ to get $\|TU-TV\|_\infty\le\gamma\|U-V\|_\infty$.

**(c)** Note $(TU)(s,a)$ ignores $(s,a)$ inside the max, so at the fixed point set $C=\max_{s',a'}U^*(s',a')$; then $U^*(s,a)=r(s,a)+\gamma C$. Maximizing both sides over $(s,a)$: $C=r_{\max}+\gamma C$ with $r_{\max}=\max_{s',a'}r(s',a')$, so $C=\frac{r_{\max}}{1-\gamma}$. Hence
$$U^*(s,a)=r(s,a)+\frac{\gamma}{1-\gamma}\,r_{\max},\qquad r_{\max}=\max_{s',a'}r(s',a').$$
(The unique fixed point exists by (b) and Banach.)

**💡 Trick.** The max inside $T$ ignores its own $(s,a)$ argument, so at the fixed point $C=\max_{s',a'}U^*$ is one scalar; maximizing $U^*(s,a)=r(s,a)+\gamma C$ over $(s,a)$ gives $C=r_{\max}+\gamma C$, i.e. $C=\frac{r_{\max}}{1-\gamma}$.

**⚠️ Watch out.** In the contraction proof you need $|\max f-\max g|\le\max|f-g|$ (not $\max f-\max g=\max(f-g)$); the $r(s,a)$ terms cancel, so only the discounted max term is bounded by $\gamma\|U-V\|_\infty$.
