## Q1 (30 pts) — Machine-maintenance discounted-cost MDP
**Topics:** MDP formalism, Bellman optimality equation, discounted cost, monotonicity of the value function, backward induction | **Pillar:** Planning | **Difficulty:** 4
**Maps to:** lecture_03, lecture_04
**Statement (English translation):**
A sequential decision problem. There is a single machine and every day one must choose an action:
- **maintenance**: the machine is sent to "maintenance". If maintenance is chosen on day $t$ then day $t$ costs $0$, day $t+1$ costs $M$, the machine returns to the "working" state, and on day $t+2$ one returns to the routine and again chooses between maintenance and continue.
- **continue**: let $k$ be the number of days since the last day the machine was in the working state. Then one of two things happens:
  1. **broken** — the machine breaks with probability $p_k$. Then the machine is sent to "repair": day $t$ costs $0$, day $t+1$ costs $B$ (with $B>M$), the machine returns to the working state, and on day $t+2$ one returns to the routine.
  2. **not broken** — the machine did not break (but does **not** return to "working"), and the number of days it has been running grows by one.

Assume the probabilities $p_k$ are non-decreasing and there exists $m\ge 1$ such that the machine breaks with certainty, $p_m=1$. Let $\pi^*$ be the optimal policy for the **discounted** return with parameter $0<\gamma<1$, and let $Q^*,V^*$ be its value functions ($\pi^*$ is optimal in the sense that it **minimizes** the value function — this is a cost problem).

a. Write the MDP of the problem formally, in which the optimal policy is well defined.

b. Write the explicit Bellman optimality equations for the given problem. (Use the optimal value function $V^*$ and the problem parameters $B,M,\gamma,\{p_k\}$.)

c. Let $k$ be the number of days since the machine was last working. Show that $\gamma B+\gamma^2 V^*(0)\ge V^*(k)$. Hint: use part (b) and the data about $B$.

d. Show that the optimal value function is non-decreasing in $k$, i.e. $V^*(k)\le V^*(k+1)$. Hint: prove by backward induction — the induction base is a state in which the machine has not been working for $m$ days.

**Solution sketch:**
**(a)** Since costs and dynamics depend only on the "age" of the machine, take the state to be $k=$ the number of days since the machine was last in the working state, $S=\{0,1,2,\dots\}$ ($k=0$ means the machine is working today). Actions $A=\{\text{maintenance},\text{continue}\}$, discount $0<\gamma<1$, and the objective is to **minimize** the expected discounted cost. The maintenance/repair "days" are folded into the transition costs (this is exactly what produces the $\gamma$ and $\gamma^2$ factors below): choosing **maintenance** at age $k$ pays a discounted $\gamma M$ and lands at age $0$ two days later; choosing **continue** at age $k$ pays nothing today and then, w.p. $p_k$, breaks (discounted cost $\gamma B$, age $0$ two days later), and w.p. $1-p_k$ moves to age $k+1$ tomorrow. Because $p_k$ is non-decreasing with $p_m=1$, the age is effectively bounded ($k\le m$), so the state space is finite and $\pi^*$ is well defined.

**(b)** Writing the two options at age $k$ and taking the cheaper one,
$$V^*(k)=\min\Big\{\underbrace{\gamma M+\gamma^2 V^*(0)}_{\text{maintenance}},\ \underbrace{p_k\big(\gamma B+\gamma^2 V^*(0)\big)+(1-p_k)\,\gamma\,V^*(k+1)}_{\text{continue}}\Big\}.$$
(Value iteration on random instances with $B>M$ and non-decreasing $p_k$ reproduces this fixed point exactly.)

**(c)** **Maintenance caps the value.** Maintenance is always an available action, so by (b) $V^*(k)\le \gamma M+\gamma^2 V^*(0)$. Since $B>M$ and $\gamma>0$,
$$V^*(k)\le \gamma M+\gamma^2 V^*(0)\;<\;\gamma B+\gamma^2 V^*(0),$$
hence $\gamma B+\gamma^2 V^*(0)\ge V^*(k)$ for every $k$. (Confirmed numerically for $400$ random instances.)

**(d)** **Backward induction that $V^*$ is non-decreasing.** Base $k=m$: since $p_m=1$, **continue** breaks for sure and costs $\gamma B+\gamma^2 V^*(0)$, whereas **maintenance** costs $\gamma M+\gamma^2 V^*(0)<$ that (as $M<B$), so $V^*(m)=\gamma M+\gamma^2 V^*(0)$ is the constant maintenance value; likewise $V^*(k)=\gamma M+\gamma^2 V^*(0)$ for all $k\ge m$, so the claim holds there. Step (from $k+1$ to $k$, $k<m$): the maintenance option $\gamma M+\gamma^2 V^*(0)$ is the same constant at every age, and the continue option $g(k)=p_k\big(\gamma B+\gamma^2 V^*(0)\big)+(1-p_k)\gamma V^*(k+1)$ is non-decreasing in $k$ — because $p_k$ is non-decreasing and, by (c), the "break" cost $\gamma B+\gamma^2 V^*(0)\ge V^*(k+1)\ge \gamma V^*(k+1)$, so raising the weight $p_k$ on the larger quantity, together with the inductive $V^*(k+1)\le V^*(k+2)$, only increases $g$. Since $V^*=\min\{\text{const},g(k)\}$ with $g$ non-decreasing, $V^*$ is non-decreasing: $V^*(k)\le V^*(k+1)$. (Verified over $400$ random instances.)

**💡 Trick.** The whole question hangs on the two-line Bellman recursion in (b): every later part just reads structure off it. Maintenance is a **constant** cost $\gamma M+\gamma^2V^*(0)$ available at every age, so it is simultaneously an upper bound (giving (c) via $B>M$) and the plateau value once $p_k=1$ (giving the base of (d)).

**⚠️ Watch out.** This is a **cost minimization**, so "optimal" means the **min**, monotonicity points the intuitive way (an older machine is more likely to break, so it costs more), and the delayed $M,B$ payments carry the $\gamma$/$\gamma^2$ discount — dropping a power of $\gamma$ is the classic error here.

## Q2 (30 pts) — Value relations under min/max of two rewards
**Topics:** value function, optimal value function, linearity of $V^\pi$ in the reward, monotonicity, min/max of rewards | **Pillar:** Planning | **Difficulty:** 4
**Maps to:** lecture_03, lecture_04
**Statement (English translation):**
Given a state set $S$, action set $A$, initial state $s_0\in S$ and transition function $p$. For a reward function $r$ let $M_r$ be the MDP $(S,A,p,s_0,r)$. For a policy $\pi$ and reward $r$, let $V^\pi_r(s)$ be the value (discounted return) of $\pi$ starting at $s$ in $M_r$; let $V^*_r(s)$ be the optimal value from $s$ in $M_r$. All rewards are in $[0,1]$, i.e. $r(s,a)\in[0,1]$, and the criterion is discounted reward with $\gamma=0.9$. Two reward functions $r_1,r_2$ are given, together with
$$\begin{aligned}r_3(s,a)&=\min\{r_1(s,a),r_2(s,a)\},\qquad\\ r_4(s,a)&=\max\{r_1(s,a),r_2(s,a)\}\quad\text{for every }s,a.\end{aligned}$$
For each part, determine the relation between the two expressions: **equal** ($=$, always the same value); **$\ge$** (the first is always $\ge$ the second, and is sometimes strictly greater); **$\le$** (the first is always $\le$ the second, sometimes strictly smaller); or **incomparable** (sometimes the first is strictly larger, sometimes the second).

a. The relation between $V^\pi_{r_3}+V^\pi_{r_4}$ and $V^\pi_{r_1}+V^\pi_{r_2}$ for every policy $\pi$.

b. The relation between $V^*_{r_3}+V^*_{r_4}$ and $V^*_{r_1}+V^*_{r_2}$.

c. The relation between $V^*_{r_1}$ and $V^*_{r_3}$.

d. Let $\pi^*_i$ be the optimal policy for reward $r_i$. If at state $s$, $\pi^*_1(s)=\pi^*_2(s)=a$ (the same action is optimal for both $r_1$ and $r_2$), must it hold that $\pi^*_3(s)=\pi^*_4(s)=a$?

**Solution sketch:**
**(a) Equal ($=$).** For a **fixed** policy $\pi$, the value is **linear** in the reward: $V^\pi_r(s)=\mathbb{E}\big[\sum_t\gamma^t r(s_t,a_t)\big]$, so $V^\pi_{r}+V^\pi_{r'}=V^\pi_{r+r'}$. Pointwise $\min+\max$ preserves the sum, $r_3+r_4=\min\{r_1,r_2\}+\max\{r_1,r_2\}=r_1+r_2$, hence
$$V^\pi_{r_3}+V^\pi_{r_4}=V^\pi_{r_3+r_4}=V^\pi_{r_1+r_2}=V^\pi_{r_1}+V^\pi_{r_2}.$$
(Verified exactly on random MDPs.)

**(b) Incomparable.** The linearity of (a) breaks once each term is separately **optimized**, because $\pi^*_{r_3},\pi^*_{r_4},\pi^*_{r_1},\pi^*_{r_2}$ can be four different policies. Both strict directions occur (exact policy-iteration witnesses):
- Strictly **$<$**: $r_1=\left(\begin{smallmatrix}0&1\\0&0\end{smallmatrix}\right)$, $r_2=\left(\begin{smallmatrix}1&0\\0&0\end{smallmatrix}\right)$ give $r_3\equiv0$ (so $V^*_{r_3}=0$) while $V^*_{r_4}\approx V^*_{r_1}\approx V^*_{r_2}$, so $V^*_{r_3}+V^*_{r_4}<V^*_{r_1}+V^*_{r_2}$.
- Strictly **$>$**: on random $3$-state MDPs the gap $(V^*_{r_3}+V^*_{r_4})-(V^*_{r_1}+V^*_{r_2})$ is positive at every state (e.g. $\approx+0.31$) when $r_1,r_2$ "disagree" about the good action, so $r_4$ exploits whichever is high.

Since both signs appear, the expressions are **incomparable**.

**(c) $V^*_{r_1}\ge V^*_{r_3}$ (i.e. $V^*_{r_3}\le V^*_{r_1}$).** $V^*_r$ is **monotone** in the reward: if $r\le r'$ pointwise then $V^*_r\le V^*_{r'}$ (any policy earns at least as much under the larger reward, so does the optimum). Here $r_3=\min\{r_1,r_2\}\le r_1$, hence $V^*_{r_3}\le V^*_{r_1}$. (Verified on $3000$ random MDPs.)

**(d) No.** Agreement of $\pi^*_1$ and $\pi^*_2$ at $s$ does **not** force $\pi^*_3,\pi^*_4$ to agree there, because $\min/\max$ reshapes the reward landscape and the discounted **future** term can flip the arg-max. Concrete counterexample ($2$ states, $2$ actions, $\gamma=0.9$): $r_1=\left(\begin{smallmatrix}0.25&0.44\\0.01&0.67\end{smallmatrix}\right)$, $r_2=\left(\begin{smallmatrix}0.45&0.43\\0.16&0.07\end{smallmatrix}\right)$ with $p(\cdot\mid s_0,\cdot)=(0.74,0.26),(0.94,0.06)$ and $p(\cdot\mid s_1,\cdot)=(0.10,0.90),(0.94,0.06)$: at $s_0$ both $\pi^*_1(s_0)=\pi^*_2(s_0)=a_1$ (strictly), yet $\pi^*_4(s_0)=a_0$. So the claim is false.

**💡 Trick.** Split the reasoning by "fixed vs. optimized policy". For a **fixed** $\pi$, $V^\pi$ is linear in $r$, so anything that is a pointwise identity of rewards ($\min+\max=r_1+r_2$) transfers verbatim (part (a)). For the **optimum**, only the direction-free facts survive — monotonicity ($r\le r'\Rightarrow V^*_r\le V^*_{r'}$, part (c)) — while identities and equalities generally fail (parts (b),(d)).

**⚠️ Watch out.** Do not carry the (a) equality into (b): once you optimize $r_3,r_4,r_1,r_2$ separately, four different policies appear and the sum is genuinely incomparable. And in (d) remember an optimal action balances immediate reward **and** discounted future — matching immediate rewards at $s$ is not enough to fix the arg-max.

## Q3 (20 pts) — Softmax (log-linear) policy gradient
**Topics:** parametric policy, softmax/log-linear policy, $\nabla\log\pi$, policy-gradient/REINFORCE | **Pillar:** Approximation | **Difficulty:** 3
**Maps to:** lecture_09
**Statement (English translation):**
Given an MDP with a finite action set $A$ and a finite state set $S$. Every state $s\in S$ and action $a\in A$ has a feature vector $\phi(s,a)\in\mathbb{R}^d$. A policy $\pi$ is parametrized by a vector $\theta\in\mathbb{R}^d$, and at state $s$ action $a\in A$ is drawn with probability
$$\pi(a\mid s;\theta)=\frac{\exp\big(\phi(s,a)\cdot\theta\big)}{\sum_{b\in A}\exp\big(\phi(s,b)\cdot\theta\big)}.$$

a. Can this policy represent the uniform random policy, which selects each of the actions with probability $1/|A|$? If yes, give a suitable $\theta$; if no, give a counterexample.

b. Compute $\nabla_\theta\log\pi(a\mid s;\theta)$.

c. Write the REINFORCE update at time $t$.

**Solution sketch:**
**(a) Yes.** Take $\theta=0$: then $\phi(s,b)\cdot\theta=0$ for all $b$, so $\pi(a\mid s;\theta)=\dfrac{e^0}{\sum_{b}e^0}=\dfrac{1}{|A|}$ at every state — exactly the uniform policy. (More generally any $\theta$ for which $\phi(s,a)\cdot\theta$ is constant over $a$ at each $s$ works.)

**(b)** Write $\log\pi(a\mid s;\theta)=\phi(s,a)\cdot\theta-\log\sum_{b}\exp(\phi(s,b)\cdot\theta)$. Differentiating, the log-partition term gives the expected feature under $\pi$:
$$\nabla_\theta\log\pi(a\mid s;\theta)=\phi(s,a)-\frac{\sum_b\exp(\phi(s,b)\cdot\theta)\,\phi(s,b)}{\sum_b\exp(\phi(s,b)\cdot\theta)}=\phi(s,a)-\mathbb{E}_{b\sim\pi(\cdot\mid s;\theta)}\big[\phi(s,b)\big].$$

**(c)** REINFORCE (Monte-Carlo policy gradient): along a sampled trajectory, for time $t$ with return $G_t=\sum_{t'\ge t}\gamma^{t'-t}r_{t'}$,
$$\theta\ \leftarrow\ \theta+\alpha\,\gamma^{t}\,G_t\,\Big(\phi(s_t,a_t)-\mathbb{E}_{b\sim\pi(\cdot\mid s_t;\theta)}\big[\phi(s_t,b)\big]\Big).$$

**💡 Trick.** The softmax score is always "observed feature minus expected feature," $\nabla_\theta\log\pi=\phi(s,a)-\mathbb{E}_{b\sim\pi}\phi(s,b)$. A positive return then pushes $\theta$ so that the taken action's feature outweighs the average, raising its probability. Uniform $=\theta$ making all logits equal, e.g. $\theta=0$.

**⚠️ Watch out.** The subtracted term is the expectation of the features $\phi(s,b)$ **under the current policy** $\pi$, not a plain average over actions and not a baseline on the return — forgetting it (writing just $\phi(s,a)$) is a common and costly slip.

## Q4 (20 pts) — Contraction of the $Q$-operator $\max\{r,\ \gamma\,\mathbb{E}\max\}$
**Topics:** Bellman/$Q$ operator, $\gamma$-contraction in sup-norm, fixed point, uniform transitions | **Pillar:** Planning | **Difficulty:** 4
**Maps to:** lecture_04, lecture_03
**Statement (English translation):**
Given an MDP $M=(S,A,p,s_0,r)$. Define the operator $T_M$ for a vector $U\in\mathbb{R}^{|S\times A|}$, with $0<\gamma<1$:
$$(T_MU)(s,a)=\max\Big\{r(s,a),\ \gamma\,\mathbb{E}_{s'\sim p(\cdot\mid s,a)}\big[\max_{a'}U(s',a')\big]\Big\}.$$

a. Write the definition of when an operator $H$ is $\gamma$-contracting for the norm $\|\cdot\|_\infty$.

b. Prove that $T_M$ is $\gamma$-contracting for the norm $\|\cdot\|_\infty$. Hint: it can be proved for each $(s,a)$ separately, splitting into cases by which term attains the max.

c. Assume that for every $(s,a)$ the transition $p(\cdot\mid s,a)$ is the **uniform** distribution over states. Show there exist constants $\rho,\theta$ such that
$$U^*(s,a)=\begin{cases}r(s,a)&\text{if }r(s,a)\ge\theta\\ \rho&\text{if }r(s,a)<\theta\end{cases}$$
and write $\rho,\theta$ as a function of $U^*(\cdot,\cdot)$ and the MDP parameters.

**Solution sketch:**
**(a)** $H$ is a $\gamma$-contraction in $\|\cdot\|_\infty$ if for all $U,U'$: $\|HU-HU'\|_\infty\le\gamma\,\|U-U'\|_\infty$.

**(b)** Use the elementary fact that "max with a fixed number" is non-expansive: for any constant $c$ and reals $x,y$, $\big|\max\{c,x\}-\max\{c,y\}\big|\le|x-y|$ (check the three cases by which branch wins). Apply it per $(s,a)$ with $c=r(s,a)$, $x=\gamma\,\mathbb{E}_{s'}\max_{a'}U(s',a')$, $y=\gamma\,\mathbb{E}_{s'}\max_{a'}U'(s',a')$:
$$\big|(T_MU)(s,a)-(T_MU')(s,a)\big|\le\big|x-y\big|=\gamma\Big|\mathbb{E}_{s'}\big[\max_{a'}U(s',a')-\max_{a'}U'(s',a')\big]\Big|.$$
Now $\big|\max_{a'}U(s',a')-\max_{a'}U'(s',a')\big|\le\max_{a'}|U(s',a')-U'(s',a')|\le\|U-U'\|_\infty$, and the expectation of something bounded by $\|U-U'\|_\infty$ is at most $\|U-U'\|_\infty$. Hence each coordinate is bounded by $\gamma\|U-U'\|_\infty$; taking the max over $(s,a)$ gives $\|T_MU-T_MU'\|_\infty\le\gamma\|U-U'\|_\infty$. By Banach, $T_M$ has a unique fixed point $U^*$.

**(c)** With $p(\cdot\mid s,a)=\text{Uniform}(S)$ for **every** $(s,a)$, the future term
$$\gamma\,\mathbb{E}_{s'\sim\text{Unif}}\big[\max_{a'}U^*(s',a')\big]=\gamma\cdot\frac{1}{|S|}\sum_{s'}\max_{a'}U^*(s',a')=:\gamma C$$
is a **constant** independent of $(s,a)$, where $C=\frac{1}{|S|}\sum_{s'}\max_{a'}U^*(s',a')$. The fixed-point equation becomes $U^*(s,a)=\max\{r(s,a),\gamma C\}$, i.e.
$$\begin{aligned}\boxed{\theta=\rho=\gamma C},\qquad\\ C=\frac{1}{|S|}\sum_{s'}\max\Big\{\max_{a}r(s',a),\ \gamma C\Big\},\end{aligned}$$
so $U^*(s,a)=r(s,a)$ when $r(s,a)\ge\gamma C$ and $U^*(s,a)=\gamma C$ otherwise — matching the required piecewise form with threshold $\theta=\gamma C$ and floor $\rho=\gamma C$. (The scalar equation for $C$ is itself a contraction in $C$, so $C$ — hence $\rho,\theta$ — is uniquely determined; verified numerically over $2000$ random uniform-transition MDPs.)

**💡 Trick.** Two moves crack it: (b) the one-line lemma $|\max\{c,x\}-\max\{c,y\}|\le|x-y|$ makes the $\max\{r,\cdot\}$ wrapper harmless, so only the $\gamma\,\mathbb{E}\max$ tail matters (and that is the usual $\gamma$-contraction). (c) **Uniform** transitions make the look-ahead term the **same constant** $\gamma C$ for all $(s,a)$, collapsing the fixed point to $U^*=\max\{r,\gamma C\}$.

**⚠️ Watch out.** The outer object is a $Q$-function on $S\times A$, and the look-ahead is $\max_{a'}U(s',a')$ (a state value), not $U(s',\cdot)$ — keep the inner $\max_{a'}$. In (c), $C$ is defined **self-consistently** through $U^*$ (it appears on both sides), so you must state the equation $C=\frac1{|S|}\sum_{s'}\max\{\max_a r(s',a),\gamma C\}$, not treat $C$ as free.
