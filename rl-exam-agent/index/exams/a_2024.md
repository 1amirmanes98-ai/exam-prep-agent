## Q1 (30 pts) — Optimal stopping: a keep-or-quit gambling game
**Topics:** MDP formalism, optimal stopping, Bellman optimality equation, monotonicity of the value function, threshold policy | **Pillar:** Planning | **Difficulty:** 3
**Maps to:** lecture_03, lecture_04
**Statement (English translation):**
A sequential decision problem with parameters $n\in\mathbb{N}$ and $p\in[0,1]$. The gambler starts the game with $0$ dollars. At every step the gambler picks one of two actions:
- **stop**: the game ends and the gambler keeps the accumulated profit.
- **continue**: with probability $p$ the gambler earns $1$ dollar (added to the accumulated profit); with probability $1-p$ the game ends and the gambler receives $0$ dollars.

The gambler can earn a total of at most $n$ dollars (i.e. if the profit reaches $n$ the gambler leaves with $n$).

a. Write the MDP of the problem formally. The rewards must be non-negative.

b. Write the explicit Bellman equations for a state whose accumulated profit is $k$ dollars. Hint: the return function is discounted with $\gamma=1$. Give $V^*(k)$.

c. Prove that $V^*(k)$ is a non-decreasing function of $k$, where $k$ is the gambler's profit. Hint: prove by backward induction over the states.

d. Prove that $V^*(k)=k$ for $k\ge p/(1-p)$, where $k$ is the gambler's profit. Hint: use the Bellman equations.

**Solution sketch:**
**(a)** State = accumulated profit $k$, so $S=\{0,1,\dots,n\}$ (plus one absorbing terminal reached by **stop** or by the $1-p$ failure). $A=\{\text{stop},\text{continue}\}$, $s_0=0$. Rewards (undiscounted, $\gamma=1$): stopping at $k$ collects $k$, i.e. $R(k,\text{stop})=k$ then absorb; continuing collects $0$ immediately, $R(k,\text{continue})=0$. Transitions from $k<n$: **stop** $\to$ absorbing; **continue** $\to k+1$ w.p. $p$ and $\to$ absorbing (profit $0$) w.p. $1-p$. At $k=n$ only **stop** is available (cap), $R(n,\text{stop})=n$. All rewards are $\ge 0$.

**(b)** Collecting the reward only at the moment of stopping, the value of holding profit $k$ is
$$V^*(k)=\max\Big\{\underbrace{k}_{\text{stop}},\ \underbrace{p\cdot V^*(k+1)+(1-p)\cdot 0}_{\text{continue}}\Big\}=\max\{\,k,\ p\,V^*(k+1)\,\},\qquad V^*(n)=n.$$
(Python backward recursion confirms, e.g. $n=4,p=0.6$: $V^*=(0.72,1.2,2,3,4)$.)

**(c)** **Backward induction that $V^*$ is non-decreasing.** Base: $V^*(n)=n\ge V^*(n-1)$ since $V^*(n-1)=\max\{n-1,pV^*(n)\}=\max\{n-1,pn\}\le n$. Step: assume $V^*(k+1)\le V^*(k+2)$. Then $V^*(k)=\max\{k,pV^*(k+1)\}$ and $V^*(k+1)=\max\{k+1,pV^*(k+2)\}$; each term of the first max is $\le$ the corresponding term of the second ($k\le k+1$ and $pV^*(k+1)\le pV^*(k+2)$), so $V^*(k)\le V^*(k+1)$. Intuition: more money is never worse — you can always **stop** and keep the extra dollar.

**(d)** **The stopping threshold.** Suppose $V^*(k+1)=k+1$ (holds at the top and propagates down while the threshold condition holds). Then **continue** is worth $p(k+1)$ and **stop** is worth $k$, so **stop** is optimal iff $k\ge p(k+1)\iff k(1-p)\ge p\iff k\ge p/(1-p)$. Hence for every $k\ge p/(1-p)$, $V^*(k)=k$ (stop), and by downward induction the assumption $V^*(k+1)=k+1$ is maintained across this whole range. (Verified: $n=10,p=0.7$, threshold $p/(1-p)=2.33$, and indeed $V^*(k)=k$ for all $k\ge3$.)

**💡 Trick.** Reward is collected only when you stop, so continuing is a pure gamble worth $p\,V^*(k+1)$ (the $1-p$ branch pays $0$): the whole question collapses to $V^*(k)=\max\{k,\,pV^*(k+1)\}$, and everything — monotonicity and the $k\ge p/(1-p)$ threshold — reads off that one equation.

**⚠️ Watch out.** The $\gamma=1$ undiscounted return is finite only because the game **terminates** (cap $n$, or the $1-p$ absorbing failure); don't add a discount. And note $V^*$ is non-**decreasing** in $k$ (more accumulated money is better) — the monotonicity direction is the opposite of the "fewer chances left" intuition.

## Q2 (30 pts) — Pairwise-max bandit feedback (MAB-max-2): explore-then-exploit regret
**Topics:** multi-armed bandits, regret minimization, explore-then-exploit, Hoeffding concentration, best-arm identification | **Pillar:** Bandits | **Difficulty:** 4
**Maps to:** lecture_10
**Statement (English translation):**
Consider a game modeled as an MAB with $k$ arms. Each arm $a_i$ has a Bernoulli reward with parameter $p_i$, with $p_1\ge\cdots\ge p_k$ and $p_k=\tfrac12$. On each round the agent chooses a **set** $S$ of $2$ of the $k$ actions, $i,j$. Rewards are sampled for each, $R_i\sim\mathrm{Ber}(p_i),R_j\sim\mathrm{Ber}(p_j)$, and the agent receives $R_S=\max\{R_i,R_j\}$. The agent only observes the combined reward $R_S$ (not the individual $R_i,R_j$). This model is called MAB-max-2.

a. For MAB-max-2, what is the set $S^*$ that maximizes the expected reward, and what is its expected reward?

For the MAB-max-2 model define an explore-exploit algorithm that takes a parameter $m$ and runs in two phases:
- explore: for each $1\le i\le k-1$, sample the set $\{i,k\}$ a total of $m$ times.
- exploit: using the samples collected in the explore phase, compute for each arm $i$ the average of the group $\{i,k\}$ and denote it $\hat p_i$. Let $i^*=\arg\max_{i\in[k-1]}\hat p_i$ and $j^*=\arg\max_{i\ne i^*}\hat p_i$; for the remaining $T-m(k-1)$ rounds play the set $\widehat{S^*}=\{i^*,j^*\}$.

b. Find $\lambda_m>0$, a function of $m,k$ and $\delta>0$, such that with probability $1-\delta$, $p_{\{i,k\}}\in[\hat p_i-\lambda_m,\hat p_i+\lambda_m]$ for each arm $i$, where $p_{\{i,k\}}$ is the expected reward of the set $\{i,k\}$.

Define the regret as the difference from the optimal policy:
$$\mathrm{REGRET}=T\cdot R(S^*)-\sum_{t=1}^{T}\mathbb{E}[R(S_t)].$$

c. Write an upper bound on the expected $\mathrm{REGRET}$ of explore-exploit, as a function of $T,k,\lambda_m,m$ and $\delta$.

d. Optimize the bound on the expected $\mathrm{REGRET}$ by fixing $m,\delta$ (the bound should depend only on $T,k$, not on $m$ or $\delta$ or $\lambda_m$). It suffices to optimize up to a constant factor.

**Solution sketch:**
**(a)** The set $\{i,j\}$ pays $\max\{R_i,R_j\}$, which is $1$ unless both draws are $0$, so its expected reward is
$$p_{\{i,j\}}=\mathbb{E}[\max\{R_i,R_j\}]=1-(1-p_i)(1-p_j).$$
This is increasing in each of $p_i,p_j$, so it is maximized by the two largest means: $\boxed{S^*=\{1,2\}}$ with $\mathbb{E}[R(S^*)]=1-(1-p_1)(1-p_2)=p_1+p_2-p_1p_2$. (Verified on $p=(.9,.8,.7,.6,.5)$: $S^*=\{1,2\}$, $\mathbb{E}[R]=0.98$.)

**(b)** Pairing arm $i$ with the reference arm $k$ (which has $p_k=\tfrac12$) gives a **Bernoulli** observation with mean $p_{\{i,k\}}=1-(1-p_i)\tfrac12=\tfrac{1+p_i}{2}$, so the $m$ i.i.d. rewards of $\{i,k\}$ have bounded range $[0,1]$ and empirical mean $\hat p_i$. By Hoeffding, $\Pr[|\hat p_i-p_{\{i,k\}}|>\lambda]\le 2e^{-2m\lambda^2}$. Union-bounding over the $k-1$ arms and setting the total failure probability to $\delta$ gives
$$\lambda_m=\sqrt{\frac{\ln\!\big(2(k-1)/\delta\big)}{2m}}.$$
Then with probability $\ge 1-\delta$, all $k-1$ estimates satisfy $p_{\{i,k\}}\in[\hat p_i-\lambda_m,\hat p_i+\lambda_m]$. Since $p_{\{i,k\}}=\tfrac{1+p_i}{2}$ is monotone in $p_i$, the ranking by $\hat p_i$ is the ranking by $p_i$ up to the $2\lambda_m$ resolution.

**(c)** Split the horizon. **Explore** uses $m(k-1)$ rounds, each losing at most $R(S^*)-R(S_t)\le 1$, contributing $\le m(k-1)$. **Exploit** runs $T-m(k-1)\le T$ rounds: on the good event (prob $\ge 1-\delta$) every $\hat p_i$ is within $\lambda_m$ of the truth, so any arm the algorithm ranks above another is at most $2\lambda_m$ worse in mean; picking the top two therefore loses at most $O(\lambda_m)$ of expected reward per round; on the bad event (prob $\le\delta$) each round loses $\le 1$. Hence
$$\mathbb{E}[\mathrm{REGRET}]\ \le\ m(k-1)\ +\ 4T\lambda_m\ +\ T\delta.$$

**(d)** Set $\delta=1/T$ so $T\delta=1$ (negligible), giving $\lambda_m=\sqrt{\ln(2(k-1)T)/(2m)}$, so
$\mathbb{E}[\mathrm{REGRET}]\lesssim m(k-1)+T\sqrt{\tfrac{\ln((k-1)T)}{m}}$. Balancing the explore term $m(k-1)$ against the exploit term $T\sqrt{\ln/m}$: differentiate and set to zero, $(k-1)=\tfrac{T}{2}\sqrt{\ln}\;m^{-3/2}$, i.e. $m\asymp\Big(\tfrac{T\sqrt{\ln((k-1)T)}}{k-1}\Big)^{2/3}$. Substituting back,
$$\mathbb{E}[\mathrm{REGRET}]=\tilde O\!\big((k-1)^{1/3}\,T^{2/3}\big)\quad(\text{with }\delta=1/T).$$
This is the classic explore-then-commit $T^{2/3}$ rate. (Simulation confirms the tradeoff: for fixed $T$, tiny $m$ mis-identifies while large $m$ wastes exploration, with the minimum in between.)

**💡 Trick.** Because the reference arm has $p_k=\tfrac12$, the pair $\{i,k\}$ is a plain Bernoulli with mean $\tfrac{1+p_i}{2}$ — monotone in $p_i$. So the exotic max-feedback reduces to ordinary mean estimation of $k-1$ Bernoullis, and the rest is textbook explore-then-commit (Hoeffding + union bound + $T^{2/3}$ balancing).

**⚠️ Watch out.** Two easy point-losses: forgetting the **union bound** over the $k-1$ arms (the $\ln$ must carry a $k$ factor, else the confidence is wrong), and choosing $\delta$ a constant — you must set $\delta=1/T$ so the $T\delta$ failure term stays $O(1)$ and doesn't dominate the $T^{2/3}$ bound.

## Q3 (20 pts) — Policy gradient for a Gumbel-distributed continuous action
**Topics:** parametric policy, $\nabla\log\pi$, policy-gradient/REINFORCE, continuous action policy | **Pillar:** Approximation | **Difficulty:** 3
**Maps to:** lecture_09
**Statement (English translation):**
The Gumbel distribution with parameter $\mu$ has density $f(a)=\exp\big(-(a-\mu)-\exp(-(a-\mu))\big)$ for all $a\in\mathbb{R}$. We use this distribution to sample actions as follows: every state $s$ has a feature vector $\phi(s)\in\mathbb{R}^d$ that characterizes it; a policy $\pi$ is parametrized by a vector $\theta\in\mathbb{R}^d$; in state $s$ actions $a\in\mathbb{R}$ are drawn from the Gumbel distribution above with parameter $\mu=\theta\cdot\phi(s)$.

a. Given a state $s$ with vector $\phi(s)$ and a policy $\pi$ parametrized by $\theta$, what is the action $a\in\mathbb{R}$ with maximal density? Prove your answer (the answer should be a function of $\phi(s)$ and $\theta$). Hint: start by exploring the one-dimensional function $\ln(f(a))$ with parameter $\mu$.

b. Compute $\nabla_\theta\log\pi(a\mid s;\theta)$.

c. Write the REINFORCE update.

**Solution sketch:**
**(a)** Let $\mu=\theta\cdot\phi(s)$ and maximize the log-density $\ell(a)=\ln f(a)=-(a-\mu)-e^{-(a-\mu)}$. Then $\ell'(a)=-1+e^{-(a-\mu)}$, which is zero iff $e^{-(a-\mu)}=1\iff a=\mu$; and $\ell''(a)=-e^{-(a-\mu)}<0$, so it is a maximum. Hence the mode is $\boxed{a^*=\mu=\theta\cdot\phi(s)}$.

**(b)** With $\mu=\theta\cdot\phi(s)$, $\log\pi(a\mid s;\theta)=-(a-\mu)-e^{-(a-\mu)}$. Differentiate through $\mu$: $\dfrac{\partial}{\partial\mu}\big[-(a-\mu)-e^{-(a-\mu)}\big]=1-e^{-(a-\mu)}$, and $\nabla_\theta\mu=\phi(s)$. Therefore
$$\nabla_\theta\log\pi(a\mid s;\theta)=\big(1-e^{-(a-\theta\cdot\phi(s))}\big)\,\phi(s).$$

**(c)** REINFORCE (Monte-Carlo policy gradient): along a sampled trajectory, for each time $t$ with return $G_t=\sum_{t'\ge t}\gamma^{t'-t}r_{t'}$,
$$\theta\ \leftarrow\ \theta+\alpha\,\gamma^{t}\,G_t\,\big(1-e^{-(a_t-\theta\cdot\phi(s_t))}\big)\,\phi(s_t).$$

**💡 Trick.** The score of any location-family density is "how far the sample sits from the mode," pushed onto the features: here $\nabla_\theta\log\pi=(1-e^{-(a-\mu)})\phi(s)$, which is positive when $a>\mu$ and negative when $a<\mu$, so a good return shifts $\mu=\theta\cdot\phi(s)$ toward the sampled action.

**⚠️ Watch out.** Don't confuse the **mode** $a^*=\mu$ (asked in (a)) with the **mean** of a Gumbel, which is $\mu+\text{(Euler's }\gamma\text{)}$ — the density peaks at $\mu$, not at its expectation. And keep the $\gamma^t$ discount in the REINFORCE step.

## Q4 (20 pts) — γ-contraction of a decoupled-reward Bellman operator
**Topics:** Bellman operator, $\gamma$-contraction in sup-norm, fixed point / optimal value | **Pillar:** Planning | **Difficulty:** 3
**Maps to:** lecture_04, lecture_03
**Statement (English translation):**
Given an MDP $M=(S,A,P,s_0,r)$. Define the operator $T$ for a vector $U\in\mathbb{R}^{|S|}$, with $0<\gamma<1$:
$$(TU)(s)=\max_a r(s,a)+\gamma\,\max_b\,\mathbb{E}_{s'\sim P(\cdot\mid s,b)}\,U(s').$$

a. Write the definition of when an operator $H$ is $\gamma$-contracting for the norm $\|\cdot\|_\infty$.

b. Prove that $T$ is $\gamma$-contracting for the norm $\|\cdot\|_\infty$.

c. Let $U^*$ be the fixed point of $T$. Show an MDP whose optimal value function is $U^*$.

**Solution sketch:**
**(a)** $H$ is a $\gamma$-contraction in $\|\cdot\|_\infty$ if for all $U,U'$: $\|HU-HU'\|_\infty\le\gamma\,\|U-U'\|_\infty$.

**(b)** The reward term $\max_a r(s,a)$ does not depend on $U$, so it cancels:
$$(TU)(s)-(TU')(s)=\gamma\Big[\max_b \mathbb{E}_{s'\sim P(\cdot\mid s,b)}U(s')-\max_b \mathbb{E}_{s'\sim P(\cdot\mid s,b)}U'(s')\Big].$$
Using $\big|\max_b x_b-\max_b y_b\big|\le\max_b|x_b-y_b|$ and $|\mathbb{E}_{s'}(U-U')|\le\|U-U'\|_\infty$,
$$|(TU)(s)-(TU')(s)|\le\gamma\max_b \mathbb{E}_{s'}|U(s')-U'(s')|\le\gamma\|U-U'\|_\infty.$$
Taking the max over $s$ gives $\|TU-TU'\|_\infty\le\gamma\|U-U'\|_\infty$. By Banach, $T$ has a unique fixed point $U^*$.

**(c)** Because the immediate reward $\max_a r(s,a)$ is decoupled from the action $b$ that drives the transition, $U^*$ is exactly the optimal value of the MDP $M'=(S',A',P',s_0',r')$ with $S'=S$, $A'=A$, $P'(s'\mid s,b)=P(s'\mid s,b)$, $s_0'=s_0$, and the **state-only** reward $r'(s,a)=\max_{a'}r(s,a')$ (independent of the action). Indeed the Bellman-optimality equation of $M'$ is $V^*_{M'}(s)=\max_b\big[r'(s)+\gamma\mathbb{E}_{s'\sim P(\cdot\mid s,b)}V^*_{M'}(s')\big]=\max_a r(s,a)+\gamma\max_b\mathbb{E}_{s'}V^*_{M'}(s')=(TV^*_{M'})(s)$, so $V^*_{M'}$ is the fixed point, i.e. $U^*=V^*_{M'}$.

**💡 Trick.** The immediate term $\max_a r(s,a)$ is a constant w.r.t. $U$, so it plays no role in the contraction — only the $\gamma\max_b\mathbb{E}$ part matters, and $\max$ + expectation are both non-expansive. Reading the operator as Bellman-optimality with a **state-only** reward $r'(s)=\max_a r(s,a)$ instantly identifies the fixed point.

**⚠️ Watch out.** The two `max`es are over **different** variables — $\max_a$ selects the immediate reward while $\max_b$ selects the transition — so this is *not* the standard $\max_a[r(s,a)+\gamma\mathbb{E}_{P(\cdot|s,a)}\dots]$; keep them separate when you build the equivalent MDP (state-only reward, free choice of transition action).
