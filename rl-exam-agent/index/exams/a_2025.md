## Q1 (30 pts) — Optimal stopping on an urn: a martingale betting game
**Topics:** MDP formalism, optimal stopping, Q-values of a fixed policy, martingale / fair game, optimal policy | **Pillar:** Planning | **Difficulty:** 4
**Maps to:** lecture_03, lecture_04
**Statement (English translation):**
A sequential decision problem. An urn holds $g$ green balls and $b$ blue balls. At each step $t$, as long as the urn has at least two balls, a ball is drawn uniformly at random from the urn. The agent then has two actions:
- **stop**: the agent earns $1$ if the drawn ball is blue and $0$ if it is green; the game then ends.
- **continue**: the agent earns $0$ and observes the color of the drawn ball; that ball leaves the urn and play continues without it.

If the urn has only a single ball, the agent earns $1$ if that ball is blue and $0$ if it is green (forced). The agent's objective is the sum of rewards until the game ends.

a. Write the MDP of the problem formally.

b. Let $\pi_{\text{cont}}$ be the policy that always plays **continue**. Compute its Q-values from the initial state. (Recall the initial state has $g$ green and $b$ blue balls.)

c. What is the optimal value function when the urn has $g'$ green and $b'$ blue balls? Hint: the recursion $f(x,y)=\tfrac{y}{x+y}f(x,y-1)+\tfrac{x}{x+y}f(x-1,y)$ with boundary $f(n,0)=0,\ f(0,n)=1$ has solution $f(x,y)=\tfrac{x}{x+y}$; the state $(b',g')$ has $b'$ blue and $g'$ green.

d. Change the **continue** action so the agent earns $q>0$ (instead of $0$). What is the optimal policy now?

**Solution sketch:**
**(a)** State = counts $(g,b)$ of green/blue balls left, $S=\{(g,b):g,b\ge0\}\cup\{\text{terminal}\}$, $s_0=(g,b)$, $A=\{\text{stop},\text{continue}\}$, undiscounted ($\gamma=1$, the game terminates). With $n=g+b\ge2$: the **expected** reward of stopping is $R((g,b),\text{stop})=\tfrac{b}{n}$ (prob. the drawn ball is blue), then $\to$ terminal; $R((g,b),\text{continue})=0$ and $P\big((g-1,b)\mid(g,b),\text{continue}\big)=\tfrac{g}{n}$, $P\big((g,b-1)\mid(g,b),\text{continue}\big)=\tfrac{b}{n}$ (the removed ball's color). At $n=1$ the forced reward is $1$ iff blue. All rewards $\ge0$.

**(b)** Stopping at $s_0$ bets on the drawn ball: $Q^{\pi_{\text{cont}}}(s_0,\text{stop})=\dfrac{b}{g+b}$. Continuing (reward $0$) and then always continuing removes balls uniformly at random until one remains, and that last ball is a uniformly random ball of the original urn, so it is blue with probability $\tfrac{b}{g+b}$: $Q^{\pi_{\text{cont}}}(s_0,\text{continue})=\dfrac{b}{g+b}$. **Both Q-values equal $\dfrac{b}{g+b}$.**

**(c)** $\boxed{V^*\big((b',g')\big)=\dfrac{b'}{b'+g'}}$ (blue-fraction). The game is a **martingale / fair game**: at any state $\text{stop}=\tfrac{b}{n}$, while

$$\begin{aligned}\text{continue}&=\tfrac{g}{n}\cdot\tfrac{b}{n-1}+\tfrac{b}{n}\cdot\tfrac{b-1}{n-1}\\ &=\tfrac{b(g+b-1)}{n(n-1)}=\tfrac{b}{n}\end{aligned}$$

too — so both actions have the same value and $V^*(g,b)=\tfrac{b}{g+b}$ for every state, matching the hint's $f(x,y)=\tfrac{x}{x+y}$ with $x=b',y=g'$. (Verified by exact DP for all $g,b\le7$: $V^*=\tfrac{b}{g+b}$ and stop $=$ continue everywhere.)

**(d)** Now every **continue** pays $q>0$ while the terminal blue-probability stays $\tfrac{b'}{b'+g'}$ regardless of when you stop (the martingale of (c)), so continuing only ever adds reward. Hence $\pi^*\big((b',g')\big)=\text{continue}$ while $\ge2$ balls remain, with $Q^*(\text{stop})=\tfrac{b'}{b'+g'}$ and $Q^*(\text{continue})=q\,(b'+g'-1)+\tfrac{b'}{b'+g'}\ge Q^*(\text{stop})$, i.e. $V^*=q\,(b'+g'-1)+\tfrac{b'}{b'+g'}$. (Verified by DP, e.g. $g'=2,b'=1,q=0.3$: $V^*=0.3\cdot2+\tfrac13=0.9\overline{3}$.)

**💡 Trick.** Recognize the **martingale**: under uniform removal the probability "the ball I eventually bet on is blue" never changes — it is $\tfrac{b}{g+b}$ from the start. So with $q=0$ every policy (stop now, or drain the urn) has the *same* value $\tfrac{b}{g+b}$; the only thing that can break the tie is a per-step bonus $q>0$, which makes **continue** strictly best.

**⚠️ Watch out.** Blue is the paying color, so the value is $\tfrac{b}{g+b}$ (blue over total) — don't flip it to $\tfrac{g}{g+b}$. And in (d), "always continue" is optimal not because it changes the win-probability (it doesn't) but purely because it harvests the extra $q$ each step; the terminal term is unchanged.

## Q2 (30 pts) — Finite-horizon reward transforms: which preserve the optimal first action
**Topics:** finite-horizon MDP, optimal policy, reward scaling/shifting, horizon dependence, action-value ordering | **Pillar:** Planning | **Difficulty:** 4
**Maps to:** lecture_02, lecture_03
**Statement (English translation):**
Let $M=(s_0,S,A,R,P)$ be a finite-horizon MDP with horizon parameter $T$. All rewards are in $[-1,+1]$ (note they can be **negative**). Let $\pi^*$ be an optimal policy for $M$ and $a_0^*$ the action it takes at the initial state $s_0$. In each part we modify $M$ and ask: **must there still be an optimal policy that takes $a_0^*$ at $s_0$?** Answer true/false with a proof or a counterexample.

a. Multiply every reward by a constant $\alpha\in[-1,+1]$.

b. Add a constant $\beta\in[-1,+1]$ to every reward.

c. Change the horizon parameter $T$ to $T'=2T$.

d. In every state $s$ add a **reset** action with reward $0$, $r(s,\text{reset})=0$, whose next state is $s_0$.

e. In some state $s$ and action $a$ with $a\ne\pi^*(s)$ and $r(s,a)\ge0$, halve that reward: $r'(s,a)=r(s,a)/2$.

**Solution sketch:**
**(a) False.** For $\alpha>0$ scaling preserves every arg-max, and for $\alpha=0$ all returns are $0$ (any action is optimal), so $a_0^*$ survives — but $\alpha\in[-1,+1]$ allows $\alpha<0$, which turns "maximize" into "minimize" and generally flips the optimal action. Counterexample: any $M$ with a unique best first action, then $\alpha=-1$ makes the worst action best, so $a_0^*$ need not stay optimal. (Confirmed: random negative-reward finite-horizon MDPs where $\alpha<0$ changes the optimal first action.)

**(b) True.** In a horizon-$T$ problem **every** policy collects exactly $T$ rewards, so adding $\beta$ to all rewards raises *every* policy's return by the same constant $\beta T$. The ordering of policies — and of first actions at $s_0$ — is unchanged, so $a_0^*$ stays optimal. (Verified: the optimal first-action set is identical for all $\beta$.)

**(c) False.** Finite-horizon optimal actions are **horizon-dependent**: the best move with $T$ steps left can differ from the best move with $2T$ steps left (e.g. a short horizon favors grabbing a near reward, a long horizon favors setting up a larger later reward). Counterexample: random MDPs where the optimal first action at $s_0$ differs between horizon $T$ and $2T$. (Confirmed numerically.)

**(d) False.** The reset action (reward $0$, back to $s_0$) lets a policy "waste" a step at value $0$. If the MDP's rewards are all **negative**, every non-reset trajectory has negative return, whereas resetting every step yields total $0$ — so the optimal first action becomes **reset**, not $a_0^*$. Counterexample: any all-negative-reward $M$. (Confirmed numerically.)

**(e) True.** The action $a$ is never used by the optimal policy at $s$, so lowering only $r(s,a)$ can neither raise the value of $\pi^*$ (which does not take $a$ at $s$) nor make $a$ attractive (it becomes even worse). Hence $V^*$ and the optimal action sets are unchanged and $a_0^*$ stays optimal. (Verified across $\sim1.6\times10^4$ MDPs where $a$ is suboptimal at $s$ for every remaining horizon: $a_0^*$ always preserved.)

**💡 Trick.** Sort the four transforms by **what they preserve**. Positive affine facts that hold *per policy* survive: adding a constant shifts every horizon-$T$ return by $\beta T$ (b), and lowering an *unused* action changes nothing (e). What breaks optimality is anything that reorders returns: negative scaling (a), a new dominating option (d, the reset), or changing how many steps you optimize over (c).

**⚠️ Watch out.** Two traps: (i) $\alpha$ and $\beta$ live in $[-1,+1]$, so you *must* consider the **negative** case in (a) (where it fails) even though it holds for $\beta$ in (b) — scaling and shifting behave differently. (ii) In (d) the reset looks harmless (reward $0$) but is decisive precisely when the original rewards are negative — always test the negative-reward regime the problem hands you.

## Q3 (20 pts) — Value-function approximation by Monte-Carlo stochastic gradient descent
**Topics:** value-function approximation, quadratic objective, gradient descent, Monte-Carlo return, stochastic gradient / SGD | **Pillar:** Approximation | **Difficulty:** 3
**Maps to:** lecture_08, lecture_06
**Statement (English translation):**
Given an MDP where every state $s$ has a feature vector $\phi(s)\in\mathbb{R}^d$. A value-function approximator with parameter $\theta\in\mathbb{R}^d$ is $\hat V(s;\theta)=\exp\big(\phi(s)\cdot\theta\big)$. For a policy $\pi$ we minimize the quadratic error
$$J^\pi(\theta)=\frac12\sum_s\mu(s)\big(V^\pi(s)-\hat V(s;\theta)\big)^2,$$
where $\mu(s)$ is the probability that policy $\pi$ reaches state $s$ (assume a state is not visited twice in the same run). We minimize $J^\pi(\theta)$ by gradient updates.

a. Write the gradient-descent update for minimizing $J^\pi(\theta)$ (as a function of $\phi(s),\mu(s),V^\pi(s)$ and learning rate $\alpha$).

b. Propose how to obtain an unbiased estimate of $V^\pi(s)$ for a state $s$, given an episode of $\pi$ in which $s$ appears, using Monte-Carlo.

c. Write the Monte-Carlo SGD update for minimizing $J^\pi(\theta)$.

**Solution sketch:**
**(a)** Since $\nabla_\theta\hat V(s;\theta)=\exp(\phi(s)\cdot\theta)\,\phi(s)=\hat V(s;\theta)\,\phi(s)$, the gradient is $\nabla_\theta J^\pi=\sum_s\mu(s)\big(\hat V(s;\theta)-V^\pi(s)\big)\hat V(s;\theta)\,\phi(s)$, so
$$\theta^{t+1}=\theta^{t}-\alpha\sum_s\mu(s)\big(\hat V(s;\theta^t)-V^\pi(s)\big)\hat V(s;\theta^t)\,\phi(s).$$
(Analytic gradient matches a finite-difference check.)

**(b)** Run the episode and, from a visit to $s$ at time $t_s$, form the **return** $G_{s}=\sum_{k\ge t_s}\gamma^{\,k-t_s}r_k$ (the discounted sum of rewards observed after $s$). The Monte-Carlo return is an **unbiased** estimate: $\mathbb{E}[G_s]=V^\pi(s)$.

**(c)** Sample states on-policy (this realizes the $\mu(s)$ weighting) and replace $V^\pi(s_t)$ by the unbiased return $G_t$. At a visited state $s_t$ with return $G_t$,
$$\theta^{t+1}=\theta^{t}-\alpha\big(\hat V(s_t;\theta^t)-G_t\big)\hat V(s_t;\theta^t)\,\phi(s_t)=\theta^{t}+\alpha\big(G_t-\hat V(s_t;\theta^t)\big)\hat V(s_t;\theta^t)\,\phi(s_t).$$

**💡 Trick.** Two facts do all the work: the chain rule on $\hat V=\exp(\phi\cdot\theta)$ gives the extra factor $\hat V(s;\theta)$ in the gradient (its own derivative), and Monte-Carlo turns the *two* intractable pieces of $J$ into samples at once — sampling states on-policy supplies $\mu$, and the return $G_t$ supplies an unbiased $V^\pi(s_t)$.

**⚠️ Watch out.** Don't drop the $\hat V(s;\theta)$ factor from $\nabla_\theta\hat V$ (the exponential differentiates to itself times $\phi$). And $G_t$ estimates $V^\pi$, not $\hat V$ — the residual is $\hat V(s_t;\theta)-G_t$; using $\hat V$ on both sides would zero the update.

## Q4 (20 pts) — γ-contraction of an operator with a global look-ahead max
**Topics:** Bellman-type operator, $\gamma$-contraction in sup-norm, fixed point, global maximum term | **Pillar:** Planning | **Difficulty:** 3
**Maps to:** lecture_04, lecture_03
**Statement (English translation):**
Given an MDP $M=(S,A,p,s_0,r)$. Define the operator $T$ for $U\in\mathbb{R}^{|S|}$, with $0<\gamma<1$:
$$(TU)(s)=\max_a\ \mathbb{E}_{s'\sim p(\cdot\mid s,a)}\Big[\,r(s,a)+\gamma\,\max_{\tilde s}\,U(\tilde s)\,\Big].$$

a. Write the definition of when an operator $H$ is $\gamma$-contracting for the norm $\|\cdot\|_\infty$.

b. Prove that $T$ is $\gamma$-contracting for the norm $\|\cdot\|_\infty$.

c. Let $U^*$ be the fixed point of $T$ for the given MDP $M$. Write its value $U^*(s)$ for every state, as a function of $\gamma$ and the reward $r(s,a)$ of $M$. Hint: think first about the state–action $(s,a)$ that maximizes $r(s,a)$.

**Solution sketch:**
**(a)** $H$ is a $\gamma$-contraction in $\|\cdot\|_\infty$ if $\|HU-HU'\|_\infty\le\gamma\|U-U'\|_\infty$ for all $U,U'$.

**(b)** The inner bracket does **not** depend on $s'$ (neither $r(s,a)$ nor $\max_{\tilde s}U(\tilde s)$ involves $s'$), so the expectation is vacuous and $(TU)(s)=\max_a r(s,a)+\gamma\max_{\tilde s}U(\tilde s)$. The reward term cancels between $U$ and $U'$:
$$(TU)(s)-(TU')(s)=\gamma\big(\max_{\tilde s}U(\tilde s)-\max_{\tilde s}U'(\tilde s)\big).$$
Since $\big|\max_{\tilde s}U(\tilde s)-\max_{\tilde s}U'(\tilde s)\big|\le\max_{\tilde s}|U(\tilde s)-U'(\tilde s)|=\|U-U'\|_\infty$, we get $|(TU)(s)-(TU')(s)|\le\gamma\|U-U'\|_\infty$ for every $s$, hence $\|TU-TU'\|_\infty\le\gamma\|U-U'\|_\infty$. By Banach there is a unique fixed point $U^*$.

**(c)** Write $R_{\max}=\max_{s',a'}r(s',a')$. The fixed point satisfies $U^*(s)=\max_a r(s,a)+\gamma\max_{\tilde s}U^*(\tilde s)$. Taking $\max_s$ of both sides, $\max_{\tilde s}U^*(\tilde s)=R_{\max}+\gamma\max_{\tilde s}U^*(\tilde s)$, so $\max_{\tilde s}U^*(\tilde s)=\dfrac{R_{\max}}{1-\gamma}$. Substituting back,
$$\boxed{U^*(s)=\max_a r(s,a)+\frac{\gamma}{1-\gamma}\,\max_{s',a'}r(s',a')}.$$
(Verified by iterating $T$ on $2000$ random MDPs.)

**💡 Trick.** Spot that the look-ahead is $\max_{\tilde s}U(\tilde s)$ — a **global** maximum over the whole vector, independent of $s$ and $s'$ — so the expectation is a red herring and the operator is just $\max_a r(s,a)$ plus a **single scalar** $\gamma\|U\|_{\max}$. That scalar solves a one-dimensional fixed point $c=R_{\max}+\gamma c$, giving the $\tfrac{\gamma}{1-\gamma}R_{\max}$ tail.

**⚠️ Watch out.** Do not read $\max_{\tilde s}U(\tilde s)$ as $\max_{a'}U(s')$ (the usual next-state look-ahead) — here it is the maximum entry of $U$ over **all** states, the same constant in every coordinate. That is exactly why $U^*$ collapses to "local best reward $+$ a global constant," rather than the standard optimal value.
