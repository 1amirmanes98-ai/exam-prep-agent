## Q1 (30 pts) — Bidirectional cycle MDP: finite horizon, policy iteration, optimal policy
**Topics:** MDP formalization, finite-horizon return, action-value $Q^\pi$, policy iteration, optimal policy, stochastic actions | **Pillar:** Planning | **Difficulty:** 4
**Maps to:** lecture_03, lecture_04
**Statement (English translation):**
An MDP is defined on a bidirectional cycle of size $2k$ with states numbered $0,\dots,2k-1$; the initial state is $s_0=k$. There are four actions:
- **NA** (null action): stay in the current state (deterministic);
- **CW** (clockwise): deterministically $i\to i+1 \bmod 2k$;
- **CCW** (counter-clockwise): deterministically $i\to i-1 \bmod 2k$;
- **J** (jump): with prob. $\tfrac12$ go $i\to i+2\bmod 2k$ and with prob. $\tfrac12$ go $i\to i-2\bmod 2k$.

The immediate reward of every state-action $(s,a)$ is $0$, except at state $0$ where action **NA** gives reward $1$ (all other actions at state $0$ give $0$). The return is **finite horizon** with horizon parameter $2k$. (The diagram for $k=3$ shows the cycle $0\text{–}1\text{–}2\text{–}3\text{–}4\text{–}5\text{–}0$.)
- **(a)** Write the MDP formally for the size-$2k$ problem: $S,A,s_0,R,P$.
- **(b)** Policy $\pi_0$ plays **CW** in every state. Write $Q^{\pi_0}(0,a)$ for state $s=0$ and each of the four actions.
- **(c)** Run Policy Iteration starting from $\pi_0$ (CW everywhere). After the first iteration we obtain $\pi_1$. What is $\pi_1$? (It suffices to give the states where $\pi_1$ and $\pi_0$ differ, the action of each, and an explanation.)
- **(d)** What is the optimal policy $\pi^*$ (for every state $s$, for the size-$2k$ MDP)?
- **(e)** Now redefine CW and CCW so that with prob. $0.9$ they behave as before and with prob. $0.1$ they stay in the current state. Does the optimal policy change? (yes/no, explain.)

**Solution sketch:**

**(a)** $S=\{0,1,\dots,2k-1\}$; $A=\{\text{NA},\text{CW},\text{CCW},\text{J}\}$; $s_0=k$; $R(s,a)=1$ if $s=0$ and $a=\text{NA}$, else $0$. Transitions: NA $\to s$ (prob 1); CW $\to (s{+}1)\bmod 2k$; CCW $\to (s{-}1)\bmod 2k$; J $\to (s{+}2)\bmod 2k$ w.p. $\tfrac12$, $(s{-}2)\bmod 2k$ w.p. $\tfrac12$. Finite horizon $H=2k$.

**(b)** Under $\pi_0$ (always CW) the agent never plays NA, so it never collects reward: $V^{\pi_0}(s)=0$ for all $s$. Hence $Q^{\pi_0}(0,a)=R(0,a)+V^{\pi_0}(\text{next})=R(0,a)$: **$Q^{\pi_0}(0,\text{NA})=1$, $Q^{\pi_0}(0,\text{CW})=Q^{\pi_0}(0,\text{CCW})=Q^{\pi_0}(0,\text{J})=0$** (python-verified).

**(c)** Policy evaluation gives $V^{\pi_0}\equiv0$, so improvement is greedy w.r.t. $Q^{\pi_0}(s,a)=R(s,a)$: $\pi_1(s)=\arg\max_a R(s,a)$. Only state $0$ has a nonzero reward (NA), all other states have $R\equiv0$ (ties, keep CW). Therefore **$\pi_1$ differs from $\pi_0$ only at state $0$: $\pi_1(0)=\text{NA}$** (vs. $\pi_0(0)=\text{CW}$); everywhere else $\pi_1=\text{CW}=\pi_0$.

**(d)** Goal: reach $0$ as fast as possible, then play NA to collect $+1$ each remaining step. Because **J advances 2 positions per step** (vs. 1 for CW/CCW), it is used near the antipode where a $\pm2$ move makes net progress, while CW/CCW are used adjacent to $0$ to land exactly on it. Backward-induction (python-verified) gives, for $k\ge3$, the (essentially stationary) optimal policy:
$$\pi^*(0)=\text{NA};\quad \pi^*(s)=\text{CCW}\ (1\le s\le k-2);\quad \pi^*(s)=\text{J}\ (s\in\{k-1,k,k+1\});\quad \pi^*(s)=\text{CW}\ (k+2\le s\le 2k-1).$$
In particular $\pi^*(s_0)=\pi^*(k)=\text{J}$. (Verified $k=2..8$; e.g. $k=3$: $[\text{NA},\text{CCW},\text{J},\text{J},\text{J},\text{CW}]$; $k=2$ collapses to $[\text{NA},\text{CCW},\text{J},\text{CW}]$.) Note this is **not** the naive shortest-deterministic-path policy — the jump action is strictly optimal in the middle band because it covers distance twice as fast.

**(e)** **No** — the optimal policy does not change (python-verified for $k=3,4,5,6$: identical policy). The $0.1$ self-loop slows CW and CCW **symmetrically** on both arcs, leaving unchanged which direction is best at each state and the J-vs-CW/CCW comparison. NA at state $0$ (deterministic reward) is unaffected. So the greedy "head toward $0$" structure is preserved.

**💡 Trick.** Since $V^{\pi_0}\equiv0$ (CW never plays NA), policy improvement is greedy on $Q^{\pi_0}=R$, which only flips state $0$. The key structural insight for $\pi^*$ is that **J covers 2 positions per step**, so it wins in the middle band $\{k-1,k,k+1\}$ while CW/CCW land exactly on $0$.

**⚠️ Watch out.** The optimal policy is **not** the naive shortest-deterministic-path policy — the jump is strictly optimal mid-cycle. In (e), the $0.1$ self-loop slows CW and CCW **symmetrically**, so it leaves $\pi^*$ unchanged.

## Q2 (30 pts) — Invariances of the optimal policy under transformations of the MDP
**Topics:** discounted MDP, optimal policy invariance, reward scaling, discount factor, initial-state independence, nonlinear reward transforms | **Pillar:** Planning | **Difficulty:** 3
**Maps to:** lecture_04, lecture_03
**Statement (English translation):**
Consider a discounted MDP $M=(s_0,S,A,R,P)$ with discount $0<\gamma<1$. In each part we modify $M$ and ask whether the optimal policy $\pi^*$ **can change** or **necessarily does not change**. If it can change, give an example; if it necessarily does not, give a short proof.
- **(a)** Multiply all rewards by a positive constant $c>0$.
- **(b)** Change the initial state $s_0$ to a different state.
- **(c)** Multiply the discount $\gamma$ by a constant $0<c<1$ (new discount $c\gamma$).
- **(d)** Square the rewards, i.e. the new reward is $r^2(s,a)$.

**Solution sketch:**

**(a) Does NOT change.** Scaling every reward by $c>0$ scales every return, so $V'_\pi=cV_\pi$ and $Q'^*(s,a)=cQ^*(s,a)$. Since $c>0$, $\arg\max_a Q$ is preserved, so $\pi^*$ is unchanged.

**(b) Does NOT change.** In a discounted MDP there exists a single policy $\pi^*$ that is optimal simultaneously from **every** starting state ($\pi^*(s)=\arg\max_a Q^*(s,a)$ for all $s$). The initial state only picks which $V^*(s_0)$ we read off; it does not alter $\pi^*$.

**(c) CAN change.** A smaller discount reweights immediate vs. future reward. Example (python-verified): at $s_0$ choose GRAB (reward $1$ now, then $0$) or WAIT (reward $0$ now, then $R=2$ next step). With $\gamma=0.9$: WAIT value $\gamma R=1.8>1\Rightarrow$ WAIT optimal. With $c=0.5$ ($c\gamma=0.45$): WAIT value $c\gamma R=0.9<1\Rightarrow$ GRAB optimal. The optimal action flips, so $\pi^*$ changes.

**(d) CAN change.** Squaring is nonlinear and does not preserve reward ordering (it makes negatives positive and distorts magnitudes). Example (python-verified): action A gives immediate reward $-2$, action B gives $+1$. Originally B is best ($1>-2$); after squaring, A gives $4$ and B gives $1$, so A becomes best. Hence $\pi^*$ changes.

**💡 Trick.** Ask whether the transform preserves the ordering of $Q^*(s,\cdot)$: a positive-scalar reward scaling ($V'=cV$) and a change of $s_0$ keep every $\arg\max_a Q^*$, so $\pi^*$ is invariant. Discount-shrinking and squaring do not.

**⚠️ Watch out.** "Can change" requires an explicit counterexample, not just intuition: a smaller $\gamma$ reweights immediate vs. future reward (WAIT flips to GRAB) and squaring turns a bad $-2$ into a winning $+4$ — nonlinear transforms break reward ordering.

## Q3 (20 pts) — REINFORCE with a Laplace-distributed policy
**Topics:** policy gradient, REINFORCE, score function, continuous-action policy | **Pillar:** Approximation | **Difficulty:** 3
**Maps to:** lecture_09
**Statement (English translation):**
The Laplace distribution with parameter $\mu$ has density $\tfrac12\exp(-|x-\mu|)$ for all $x$. It is used to model actions: every state $s$ has a feature vector $\phi(s)\in\mathbb R^d$; the policy $\pi$ is characterized by $\theta\in\mathbb R^d$; and in state $s$, actions $a\ge0$ are drawn from the Laplace distribution with parameter $\mu=\theta\cdot\phi(s)$.
- **(a)** What is the probability (density) that action $a\ge0$ is drawn in state $s$ (with $\mu=\theta\cdot\phi(s)$)?
- **(b)** Compute $\nabla_\theta\log\pi(a\mid s;\theta)$.
- **(c)** Write the REINFORCE update for the Laplace distribution.

**Solution sketch:**

**(a)** $\pi(a\mid s;\theta)=\tfrac12\exp\!\big(-|a-\theta\cdot\phi(s)|\big)$ for $a\ge0$.

**(b)** $\log\pi=\log\tfrac12-|a-\theta\cdot\phi(s)|$, and $\nabla_\theta|a-\theta\cdot\phi(s)|=-\operatorname{sign}(a-\theta\cdot\phi(s))\phi(s)$, so
$\nabla_\theta\log\pi(a\mid s;\theta)=\operatorname{sign}\!\big(a-\theta\cdot\phi(s)\big)\,\phi(s)$ ($+\phi(s)$ if $a>\theta\cdot\phi(s)$, $-\phi(s)$ if $a<\theta\cdot\phi(s)$).

**(c)** REINFORCE: $\theta^{t+1}=\theta^{t}+\alpha\,G_t\,\nabla_\theta\log\pi(a_t\mid s_t;\theta)=\theta^{t}+\alpha\,G_t\,\operatorname{sign}\!\big(a_t-\theta\cdot\phi(s_t)\big)\phi(s_t)$, where $G_t=\sum_{k\ge0}\gamma^k r_{t+k}$ is the return from time $t$ (optionally with a baseline / $\gamma^t$ weighting).

**💡 Trick.** The whole question hinges on the Laplace score $\nabla_\theta\log\pi=\operatorname{sign}(a-\theta\cdot\phi(s))\,\phi(s)$; the $\log\tfrac12$ constant drops and only the $|a-\theta\cdot\phi|$ term survives.

**⚠️ Watch out.** Get the sign right: $\nabla_\theta|a-\theta\cdot\phi|=-\operatorname{sign}(a-\theta\cdot\phi)\phi$, so the two minus signs cancel to give $+\operatorname{sign}(a-\theta\cdot\phi)\phi$ in $\nabla_\theta\log\pi$.

## Q4 (20 pts) — Observation-policy Bellman operator for a POMDP: γ-contraction and fixed point
**Topics:** POMDP, observation-based policy, Bellman operator, γ-contraction, sup-norm, fixed point, value function | **Pillar:** Planning | **Difficulty:** 3
**Maps to:** lecture_04, lecture_03
**Statement (English translation):**
Given a POMDP $M=(S,A,p,s_0,R,O,Ob)$, define for $U\in\mathbb R^{|S|\times|O|}$ and an observation-based policy $\pi:O\to A$ (the policy depends only on the last observation; assume $O$ is finite) the operator
$$(TU)(s,o)=r\big(s,\pi(o)\big)+\gamma\,\mathbb E_{s'\sim p(\cdot\mid s,\pi(o))}\,\mathbb E_{o'\sim Ob(\cdot\mid s',\pi(o))}\big[U(s',o')\big].$$
- **(a)** State the definition of an operator $H$ being $\gamma$-contracting with respect to $\lVert\cdot\rVert_\infty$.
- **(b)** Prove that $T$ is $\gamma$-contracting with respect to $\lVert\cdot\rVert_\infty$.
- **(c)** What does the fixed point $U$ of $T$ ($TU=U$) represent for the policy $\pi$ (e.g. one of $V^*,Q^*,V^\pi,Q^\pi$)?

**Solution sketch:**

**(a)** $H$ is $\gamma$-contracting in $\lVert\cdot\rVert_\infty$ iff for all $U,V$: $\lVert HU-HV\rVert_\infty\le\gamma\lVert U-V\rVert_\infty$.

**(b)** The reward term $r(s,\pi(o))$ is identical in $TU$ and $TV$ and cancels; $T$ has no $\max$ (the action is fixed by $\pi(o)$). Hence for every $(s,o)$:
$|(TU)(s,o)-(TV)(s,o)|=\gamma\big|\mathbb E_{s',o'}[U(s',o')-V(s',o')]\big|\le\gamma\,\mathbb E_{s',o'}|U(s',o')-V(s',o')|\le\gamma\lVert U-V\rVert_\infty.$
Taking the max over $(s,o)$: $\lVert TU-TV\rVert_\infty\le\gamma\lVert U-V\rVert_\infty$. $\square$

**(c)** The fixed point $U(s,o)$ satisfies $U(s,o)=r(s,\pi(o))+\gamma\,\mathbb E_{s'}\mathbb E_{o'}[U(s',o')]$, i.e. it is the **value function $V^\pi$ of the observation-based policy $\pi$**, indexed by the (true state, current observation) pair: $U(s,o)=V^\pi(s,o)$ = the expected discounted return of following $\pi$ starting from true state $s$ having just observed $o$. (Since $T$ is a $\gamma$-contraction it has a unique fixed point, which is exactly this $V^\pi$.)

**💡 Trick.** Because $\pi(o)$ fixes the action, $T$ has **no $\max$** and the reward $r(s,\pi(o))$ cancels between $TU$ and $TV$, leaving only the $\gamma\,\mathbb E_{s',o'}$ term — the contraction is immediate and the fixed point is the policy's own value.

**⚠️ Watch out.** The fixed point is $V^\pi$ (a fixed observation-policy), **not** $V^*$ or $Q^*$; note it is indexed by the (true state, current observation) pair $(s,o)$, since $\pi$ acts only on $o$.
